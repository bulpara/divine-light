## Tech Stack

Intentionally minimal:

- Pure HTML, CSS, and vanilla JavaScript
- No frameworks, no build tools, no dependencies
- Single file (`divine-light.html`)
- Google Fonts: Cormorant Garamond, Amiri, Noto Naskh Arabic
- Fully responsive
- Works offline once loaded

This was a deliberate choice. A spiritual reading tool shouldn't require `npm install`. Just open the file.
